 ## Цель
Провести анализ внутриигровых действий пользователей и маркетинговых затрат. Вывдвинуть предложения по формированию модели монетизации мобильной игры.
 ## Результат
Проведен исследовательский анализ внутриигрового пользовательского поведения, рассчитаны DAU, CAC, Retention и другие качественные метрики источников трафика. Выдвинуто предложение по размещению рекламы на странице постройки объекта с минимальной стоимостью показа 0.06 y.e.

 ## Использованные библиотеки
Pandas, NumPy, Matplotlib, Seaborn, Plotly, SciPy
