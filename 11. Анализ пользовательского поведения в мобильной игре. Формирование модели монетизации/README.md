 ## Задачи
Провести анализ внутриигровых действий пользователей и маркетинговых затрат. Вывдвинуть предложения по формированию модели монетизации мобильной игры.
 ## Что сделано
Для решения задачи провел исследовательский анализ внутриигрового пользовательского поведения, анализ бизнес-показателей(DAU, CAC, Retention), а так же дополнительно ввел качественные метрики рекламных источников, из которых приходят пользователи. Проверил ряд статистических гипотез, после чего сформировал предложения по формированию модели монетизации. Дополнительно к основной задаче построил дашборд в Tableau.
