 ## Задачи
Выполнить приоритизацию гипотез по увеличению выручки интернет-магазина, проанализировать результаты A/B теста и принять решение о его дальнейшей судьбе.

 ## Результат
Принято решение об остановке теста. Обнаружена статистически значимая разница между конверсиями тестовой и контрольной групп. Зафиксирована победа тестовой группы, относительный прирост конверсии составил 14%.

 ## Использованные библиотеки
 Pandas, NumPy, Matplotlib, Seaborn, Plotly, SciPy
